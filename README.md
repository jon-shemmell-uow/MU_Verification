
# MU Gold Standard Pipeline

This repository contains the MU Gold Standard Pipeline, a specialized application for analyzing EMG and Torque data, performing spike timing analysis, and auditing motor unit (MU) data.

## Getting Started

### Prerequisites
- Python 3.x
- Git

### Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/jon-shemmell-uow/MU_Verification/tree/main
    cd MU_Verification
    ```

2. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the application**:
    ```bash
    python main.py
    ```

## Features

- EMG and torque data analysis
- Spike timing analysis
- Motor unit auditing and validation
- Data visualization
